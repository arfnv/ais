Overview

The trim function removes any leading or trailing spaces from a String or Text, and removes extra spaces from a Text or String, leaving one space separating each word.

Usage

The trim function is used to remove excess spaces from a String or Text.

 

Syntax

Expression:

(trim string)

(trim text)

Arguments Name Type Description
Argument:stringString or Text Any String or Text or cell or cell reference that evaluates to a String or Text for which you want to trim extra spaces.

Returns:

The result string after trimming
