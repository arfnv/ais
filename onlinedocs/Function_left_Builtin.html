Overview

The left function returns a specified number of characters from a Text or String argument, beginning at the leftmost character.

Usage

The left function is used to create a substring of a specified length starting from the leftmost character in a String or Text.

 

Syntax

Expression:

(left text length)


Arguments Name Type Description
Argument:text String or Text A String object or Text
Argument:lengthInteger A positive integer representing the number of characters to return.

Returns:

The substring of length {length} starting from the leftmost character
