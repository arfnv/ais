Overview

The getGlobalValue function returns the global value of the specified Symbol argument. Also the ^ operator in front of a Symbol returns the global value of the specified Symbol.

Usage

The getGlobalValue function is used to retrieve a Symbol's value.

 

Syntax

Expression:

(getGlobalValue symbol)

Arguments Name Type Description
Argument:symbolSymbol a Symbol

Returns:

The Symbol's value
