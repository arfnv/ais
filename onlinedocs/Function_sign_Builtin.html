Overview

The sign function determines whether the argument is positive or negative.

Usage

The sign function is used to determine if a number is positive or negative.

 

Syntax

Expression:

(sign num)

Arguments Name Type Description
Argument:numNumber A Number

Returns:

Returns 1 if the argument is positive and -1 if the argument is negative.
