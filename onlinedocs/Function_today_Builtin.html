Overview

The today function returns the Julian date value for the current computer system date. The Julian date values is automatically rounded to the last whole day.

Usage

The today function provides the current date using the computer's clock. There are no arguments to this function.

 

Syntax

Expression:

(today)


Arguments Name Type Description
Argument:noneInteger No arguments

Returns:

The current Julian date based on the computer's system date.
