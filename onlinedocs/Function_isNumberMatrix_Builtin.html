Overview

The isNumberMatrix function returns true if the argument {obj} is a Number Matrix; otherwise, false is returned.

Usage

The isNumberMatrix function is a Boolean function that determines if an object is a Number Matrix.

 

Syntax

Expression:

(isNumberMatrix obj )


Arguments Name Type Description
Argument:objCharacter, String, Symbol, Vector, BitVector, ByteVector, IntVector, FltVector, ObjVector, Structure, Dictionary, Directory, Matrix, NumMatrix, Integer, Float, Number, Complex, Boolean An Analytic Object

Returns:

Returns true if the object is a Number Matrix, otherwise false is returned.
